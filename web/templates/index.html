Traffic Limiter

Designed for unlimited configs with fair usage. When you don't want to set an explicit data_limit on configs but still need to prevent abuse, this feature runs in the background to monitor each user's consumed traffic. If a user exceeds the configured threshold (default 250 GB), their account is automatically disabled and you receive a Telegram notification with an inline Exempt button. Exempt users are never disabled.
